# AWS Cloud Practitioner Flashcards

An interactive flashcard web application to help you prepare for the AWS Certified Cloud Practitioner (CLF-C02) exam.

## Features

- **100+ Comprehensive Flashcards** covering all exam domains
- **Category Filtering** to focus on specific topics
- **Progress Tracking** to monitor your study progress
- **Keyboard Shortcuts** for efficient studying
- **Responsive Design** works on desktop and mobile
- **Shuffle Mode** to randomize card order

## Exam Coverage

The flashcards cover all four domains of the AWS Certified Cloud Practitioner exam:

1. **Cloud Concepts (24%)** - Cloud computing models, deployment models, AWS advantages, Well-Architected Framework
2. **Security & Compliance (30%)** - Shared Responsibility Model, IAM, security services, compliance
3. **Technology & Services (34%)** - Compute, storage, database, networking, and other AWS services
4. **Billing, Pricing & Support (12%)** - Pricing models, cost management tools, support plans

## How to Use

1. Open `index.html` in your web browser
2. Click on a card to flip and reveal the answer
3. Use the navigation buttons to move between cards
4. Filter by category to focus on specific topics
5. Click "Shuffle" to randomize the deck

## Keyboard Shortcuts

- **Space** - Flip current card
- **Left Arrow** - Previous card
- **Right Arrow** - Next card
- **S** - Shuffle cards

## Topics Covered

### Cloud Concepts
- Advantages of cloud computing
- Cloud deployment models (Public, Private, Hybrid)
- Cloud service models (IaaS, PaaS, SaaS)
- AWS Well-Architected Framework (6 pillars)
- Elasticity, scalability, high availability

### Security & Compliance
- AWS Shared Responsibility Model
- IAM (Users, Groups, Roles, Policies)
- Security services (CloudTrail, Config, GuardDuty, Shield, WAF)
- Best practices (MFA, least privilege)

### Technology & Services
- **Compute**: EC2, Lambda, Elastic Beanstalk, Lightsail
- **Storage**: S3, EBS, EFS, Glacier
- **Database**: RDS, DynamoDB, Aurora, Redshift
- **Networking**: VPC, CloudFront, Route 53, ELB
- **Management**: CloudWatch, Auto Scaling, CloudFormation
- **Messaging**: SNS, SQS
- **Containers**: ECS, EKS, Fargate
- **Global Infrastructure**: Regions, Availability Zones, Edge Locations

### Billing, Pricing & Support
- EC2 pricing models (On-Demand, Reserved, Spot, Savings Plans)
- AWS Free Tier
- Cost management tools (Cost Explorer, Budgets, Pricing Calculator)
- Support plans (Basic, Developer, Business, Enterprise)
- Trusted Advisor
- AWS Organizations and Consolidated Billing

## Study Tips

1. Go through all cards at least once to get familiar with the content
2. Use category filtering to focus on your weak areas
3. Shuffle regularly to avoid memorizing the order
4. Review cards multiple times until you can answer without flipping
5. Combine with practice exams and official AWS documentation
